Product wordmarks for the main display's white box.

Provide one file per product, TRANSPARENT background, tightly cropped to the text:
  atogen.png
  pantogen.png
  vertihist.png
  celomet-sr.png
  empabest.png

PNG (transparent, >=1500px wide) or SVG both work. If you use SVG, name them
.svg and update productSrc() in src/lib/products.ts. Until a file is present a
styled-text fallback is shown. Add/rename products in src/lib/products.ts.
